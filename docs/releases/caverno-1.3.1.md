# Caverno 1.3.1

Initial direct-distribution macOS release for the Sparkle update lane.

## Changes

- Add Sparkle-based update checks for direct macOS distribution.
- Package the macOS app as a notarized Developer ID release artifact.
- Publish update metadata through the Caverno macOS S3 appcast bucket.

## Notes

- Automatic update checks are enabled.
- Automatic downloads and silent installs remain disabled.
- Manual update checks are available from Advanced Debug settings.
