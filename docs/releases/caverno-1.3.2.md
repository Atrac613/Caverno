# Caverno 1.3.2

Routines workspace usability update.

## Changes

- Rework the routines workspace into a master-detail layout that matches the chat and coding workspaces: the left pane lists routines with a create button, and the right pane shows the selected routine or the routines dashboard.
- Move each routine's enable/disable switch onto its detail screen.
- Slim the routines home view into a read-only dashboard.

## Notes

- Automatic update checks are enabled.
- Automatic downloads and silent installs remain disabled.
- Manual update checks are available from Advanced Debug settings.
